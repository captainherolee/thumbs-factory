# thumbs-factory

The solution is for monitoring of smart factory regarding the usage of electricity and gas.

## thumbs-web

### Significant Library

- React / typescript
- recoil
- axios
- material UI
  There are 3 tabs in SPA.

## thumbs-server

### Significant Library

- nodeJS
- express
- MariaDB connector
- dotenv

### API

- GET /ping
- GET /electricity
- GET /electricity/status
- GET /gas
- GET /gas/status
- GET /log
