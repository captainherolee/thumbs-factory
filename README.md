# thumbs-factory

The solution is for monitoring of smart factory regarding the usage of electricity and gas.

## thumbs-web

### Software Requirement

- React / typescript
- recoil
- axios
- material UI

## thumbs-server

### Software Requirement

- nodeJS >= v16
- express
- MariaDB connector
- dotenv

### API

- GET /ping
- GET /electricity
- GET /electricity/status
- GET /gas
- GET /gas/status
- GET /log

## How To Run

- docker-compose up -d
