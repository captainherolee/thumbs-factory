# thumbs-factory

The solution is for monitoring of smart factory regarding the usage of electricity and gas.

## thumbs-web

### Software Requirement

- React / typescript
- recoil
- axios
- material UI

## thumbs-server

### Software Requirement

- nodeJS >= v16
- express
- MariaDB connector
- dotenv

### API

- GET /api/v1/ping
- GET /api/v1/electricity
- GET /api/v1/electricity/status
- GET /api/v1/gas
- GET /api/v1/gas/status
- GET /api/v1/log

## How To Run

- docker-compose up -d
